This folder contains example visualizations generated using the spatial analysis scripts from this repository.

**Shot chart visualization (*shot_attempts.png*)**

Shows spatial distribution of all shot attempts and their outcomes, revealing shooting tendencies and success patterns across different court zones.

Green circles: Made shots

Red crosses: Missed shots


**Points per Attempt visualization (*points_per_attempt.png*)**

Hexagonal grid visualization showing Points per Attempt (PPA) across the court. Colors indicate shooting efficiency.

This map reveals the team's spatial shooting patterns and identifies high-value shooting locations.
